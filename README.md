# GuiaPerguntas
o Guia perguntas é uma plataforma de perguntas e resposta onde as pessoas podem fazer suas perguntas e outras posam tirar suas duvidas!

### Tecnologias utilizadas:
- Ejs
- Css
- NodeJs
- MySQL
